Desi Drip Website

This repository contains the source code for Desi Drip, a single-page website showcasing a fusion of traditional Indian design elements with contemporary fashion aesthetics. The project is built as a clean, responsive HTML template featuring a gallery, collection overview, custom order modal, and contact information.

Project Overview

The Desi Drip website is designed as a modern, minimal, and mobile-friendly landing page for a conceptual fashion brand. It provides:

A hero section introducing the brand

A styled collection overview

An interactive image gallery

A custom order form with modal interface

Contact and footer sections

The entire layout is built using pure HTML, CSS, and JavaScript, without external frameworks or libraries.

Features

Fully responsive single-page design

Smooth scroll navigation

Reveal-on-scroll animations

Image hover effects

Custom order form with validation

Lightweight and dependency-free

Organized color variables and reusable styling elements

Deployment

To deploy via GitHub Pages:

Go to Settings in your repository

Navigate to Pages

Under Build and Deployment, select:

Branch: main

Folder: / (root)

Save changes

Your website will be available at:

https://<your-username>.github.io/<repository-name>

Usage

Clone the repository and open the HTML file in any browser:

git clone https://github.com/<your-username>/<repository-name>.git
cd <repository-name>
open index.html


You may customize:

Brand name and logos

Color palette (defined in :root)

Images in the gallery

Text content in each section

Contact details

Order form fields and logic
